## How to run
Prerequisites:
- PHP 8.2 or above
- Composer
- Docker (optional)

First run `composer install` to add vendor files.
Afterward, run `php artisan serve` to serve the application on port 8000.
Optionally, you can run it docker using `./vendor/bin/sail up`.
